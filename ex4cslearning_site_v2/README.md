# Ex4CSLearning — Pilot Website (Colorful Edition)

This folder is ready for **GitHub Pages** with colorful, student‑friendly styling, YouTube link slots, and a Mapping page.

## Publish steps
1) Push folder to your GitHub repo (e.g., `Ex4CSLearning`).  
2) Repo **Settings → Pages** → Source: **Deploy from a branch**; Branch: `main`; Folder: `/docs`.  
3) Edit `docs/_config.yml` with your **Qualtrics**, **CTFd**, and **YouTube** links.  
4) (Optional) Upload `docs/assets/img/logo.png` and `docs/assets/img/favicon.png` for branding.

## Pages
- `index.md` — home with Quick Links
- `osint.md` — Day 1, with YouTube section
- `password-cracking.md` — Day 2, with YouTube section
- `cryptography.md` — Day 3, with YouTube section
- `resources.md` — rules & tips
- `mapping.md` — standards mapping tables
- `_layouts/default.html` — colorful header, hero, nav
- `assets/style.css` — theme styles

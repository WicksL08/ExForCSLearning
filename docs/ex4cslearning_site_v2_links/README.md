# Ex4CSLearning — Pilot Website (Colorful + YouTube)

This folder is ready for **GitHub Pages** with colorful styling, YouTube link slots, and a Mapping page.

## Publish steps
1) Push folder to your GitHub repo (e.g., `Ex4CSLearning`).  
2) Repo **Settings → Pages** → Source: **Deploy from a branch**; Branch: `main`; Folder: `/docs`.  
3) Edit `docs/_config.yml` with your **Qualtrics**, **CTFd**, and **YouTube** links (already added).  
4) (Optional) Upload `docs/assets/img/logo.png` and `docs/assets/img/favicon.png` for branding.
